# 中文微博情感数据库（2分类数据集）

## 介绍

本仓库提供了一个中文微博情感数据库，适用于情感分析任务中的二分类问题。该数据集包含了带情感标注的微博语料，共计10500条数据，其中训练集10000条，测试集500条。

## 数据集描述

### 数据格式

- **文件结构**: 数据集分为两个文件：`train.txt`（训练集）和`test.txt`（测试集）。
- **每条语料格式**: 每条语料的格式如下：
  ```
  mid 情感标签 微博文本
  ```
  - `mid`: 每条微博的唯一标签，可以通过`https://m.weibo.cn/status/ + mid`访问到该条微博的网页（部分微博可能已被博主删除）。
  - `情感标签`: 0表示负面情感，1表示正面情感。
  - `微博文本`: 微博的实际内容，其中微博表情、话题、地理定位、视频、文本超链接等都已转义成特定格式。

### 数据处理

- **微博表情**: 微博表情被转义成`[xx]`的格式，例如`[doge]`、`[允悲]`。
- **特殊格式**: 微博话题、地理定位、视频、文本超链接等被转义成`{%xxxx%}`的格式，使用正则表达式可以很方便地进行清洗。

## 使用说明

1. **数据下载**: 直接下载本仓库中的`train.txt`和`test.txt`文件即可使用。
2. **数据清洗**: 由于微博文本中包含多种转义格式，建议使用正则表达式进行数据清洗，以便更好地提取和处理文本信息。
3. **情感分析**: 该数据集适用于情感分析任务，特别是二分类问题，可以用于训练和评估情感分析模型。

## 注意事项

- 部分微博可能已被博主删除，因此通过`mid`访问微博网页时可能会出现404错误。
- 数据集中的情感标签为二分类，0表示负面情感，1表示正面情感。
