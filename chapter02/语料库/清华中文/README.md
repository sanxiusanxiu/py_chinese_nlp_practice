# 清华大学开放中文词库

THUOCL（THU Open Chinese Lexicon）是由清华大学自然语言处理与社会人文计算实验室整理推出的一套高质量的中文词库。本词库旨在为中文自然语言处理领域提供有力支持，其词表来源于主流网站的社会标签、搜索热词、输入法词库等。

## 词库特点

1. 包含词频统计信息DF值（Document Frequency），方便用户根据需求进行个性化选择。
2. 经过多轮人工筛选，保证词库收录的准确性。
3. 开放更新，我们将不断更新现有词表，并推出更多类别词表。
4. 欢迎专业人士加入，协作建设开放词库。

## 使用场景

该词库可以用于中文自动分词，提升中文分词效果。建议搭配本组研制开发的THULAC工具包使用，以提升特定领域中文分词的效果。
